# generalServer(v1.0)
A generalServer

## 技术栈

- **前端**：Vant(UI)，vue全家桶

- **后端**：Node，Express框架，高德地图API

- **数据库**：Mongodb


# 目标功能

- 登录 && 注册：

    - [x] 账号密码 -- 完成

    - [ ] 手机验证码

    - [ ] 第三方登录

## 项目运行

``` bash
# 克隆项目
git clone git@github.com:Bottle0709/generalServer.git

npm install

# 打开端口localhost:8008
npm run serve

# 前端项目打包
npm run build

```
